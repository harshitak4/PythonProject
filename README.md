# PythonProject

A collection of beginner-friendly Python programs created to practice and demonstrate core Python programming concepts such as variables, conditionals, loops, functions, and user input handling.

This repository contains simple command-line applications suitable for beginners and students learning Python.

---

## Projects Included

### Calculator
A command-line calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

File: calculator.py

---

### Password Generator
A program that generates strong and random passwords using:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

File: passwordgenerator.py

---

### Rock Paper Scissors
An interactive Rock-Paper-Scissors game where the user plays against the computer.

Features:
- Random computer choice
- Win / Lose / Draw logic
- User input validation

File: rockpaper.py

---

### To-Do List Application
A simple command-line based To-Do List application to manage daily tasks.

Features:
- Add tasks
- View tasks
- Delete tasks

File: todo.py

---

## Getting Started

### Prerequisites
- Python 3.x installed on your system

Check Python version:
```bash
python --version
```

---

### How to Run

Clone the repository:
```bash
git clone https://github.com/harshitak4/PythonProject.git
```

Navigate to the project folder:
```bash
cd PythonProject
```

Run any program:
```bash
python calculator.py
```

Replace calculator.py with the desired file name.
